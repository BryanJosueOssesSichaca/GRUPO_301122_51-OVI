La primera impresión da mucho que decir de algo. Todo entra por los ojos y lo procesa el cerebro. Esto mismo se aplica al desarrollo web, si una página web es simple, no tiene colores, los botones son estáticos y con el contenido dan ganas de salir de ahí lo más pronto posible, es seguro que no se está haciendo una buena gestión y esto llevara al fracaso, pero en cambio, si la página está bien estilizada, con colores bonitos y apariencia agradable, los usuarios se sentirán a gusto con ella y no querrán salir de ella, con lo que se asegura éxito rotundo.


CSS se dedica a eso y mucho más, la apariencia lo es todo y este lenguaje de Hojas de estilo en cascada brinda infinidad de elementos que pueden ocupar los programadores para dar a conocer al mundo un proyecto web de maravilla. CSS trabaja de la mano con HTML porque ¿cómo pintas, amueblas y decoras una casa si esta no está construida?, esta es una analogía válida para dar a entender lo que se pretende explicar.
CSS ha evolucionado el mundo de desarrollo web, ya en su tercera versión, cuenta con mucho terreno por recorrer para nunca dejar de sorprender.
