Actividad Git y Github

Vista previa - Actividad Git y Github (12 de Abril del 2018) “elaboración propia - Bryan Josue Osses Sichaca”

La actividad de Git y Github consiste en un cuestionario que contiene preguntas de respuestas multiples y unicas . las preguntas de respuestas unicas otorgan un 1 punto por acierto mientras que las preguntas de respuestas multiples otorgan 0.5 puntos por respuesta acertada (de seleccionar mas de 2 opciones la pregunta es anulada y otorga 0 puntos).
