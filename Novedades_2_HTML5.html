FORMULARIOS


Los formularios son ocupados bastante en las paginas web al momento de requerir informacion para luego ser guardada en una base de datos.
Los elementos y atributos son mas semanticos y se hace mas facil el diseñarlos. Las funcionalidades brindan una experiencia diferente para los usuarios al permitir se comporten de la misma forma en diferentes sitios web y que de manera instantanea se devuelva la informacion ingresada.

Propiedades


Label.control

Devuelve el controlador para quien está hecha la etiqueta, determinado por el atributo for.
Por ejemplo se tiene un formulario con dos cajas de texto, una para el primer nombre y otra para el primer apellido, a traves del atributo for se asocia el control con su etiqueta correspondiente.
Tambien es valido colocar dentro de la etiqueta label el control y se obtendra el mismo resultado.

< ! - - Utilizando la etiqueta for - - >

< ! - - Primer Nombre - - >
< label for = "P_Nombre" > Primer Nombre: < /label >
< input type = "text" id = "P_Nombre" name = "P_Nombre" />

< ! - - Primer Apellido - - >
< label for = "P_Apellido" > Primer Apellido: < /label >
< input type = "text" id = "P_Apellido" name = "P_Apellido" />

< ! - - Colocando control dentro de label - - >
< label > Primer Nombre: < input type = "text" id = "p_nombre" autofocus /> < /label >
< label > Primer Apellido: < input type = "text" id = "p_apellido" /> < /label >

Validacion Restringida


Dentro de los formularios siempre se deben validar los campos y se restringe el tipo de datos, la cantidad de caracteres, el formato y demas.
Dentro de las restricciones se encuentran:

Pattern

El dato ingresado debe coincidir con unos parametros en especifico.

< input type = "text" name = "Celular" pattern = "[0-9]{11}" title = "11 digitos para telefono celular" >


Restriccion Parttern (14 de Abril del 2018) “elaboración propia”


Min y Max

La cantidad minima y maxima de datos ingresados.

< input type = "range" name = "rango" min = "2" max = "10" title = "Rango desde 2 hasta 10" >


Restriccion Min y Max (14 de Abril del 2018) “elaboración propia”


Step

Al combinarse con los atributos anteriores establece un intervalo de datos permitidos.

< input type = "number" name = "puntaje" step = "2" title = "Puntaje en multiplos de 2" >


Restriccion Step (14 de Abril del 2018) “elaboración propia”


MaxLength

La cantidad maxima de caracteres ingresados.

< input type = "password" name = "clave" maxLength = "8" title = "8 caracteres maximo para la clave" >


Restriccion Maxlength (14 de Abril del 2018) “elaboración propia”


Required

Establece un campo obligatorio.

< input type = "text" name = "cedula" required title = "Campo de cedula obligatorio" >

RESPONSIVE


Responsive se refiere a la adaptabildiad de la pagina web a diferentes resoluciones de pantalla. Esto se implemento para que la pagina pueda visualizarse en diferentes dispositivos: computadores, tablets, smartphones y otros.
HTML5 se relaciona con el diseño web responsive de la siguiente manera:

  1. Se mantienen en el navegador.
  2. Son independientes de cualquier plataforma.

El diseño web responsive es flexible y HTML5 con sus nuevas funcionalidades brinda un codigo liviano que permite una interpretacion de codigo mas eficaz y por ende las paginas que se abran por ejemplo en cualquier dispositivo movil cargaran rapido.


CANVAS


Es un contenedor usado para dibujar gráficos en una página web con el uso de JavaScript.

Cuenta con varios métodos para dibujar rutas, cuadros, círculos, texto y agregar imágenes.
El tamaño por defecto es de 300 pixeles de ancho por 150 pixeles de alto.
A continuacion se muestra un ejemplo de un canvas basico, adandole un borde con estilos css.

< canvas id = "MiCanvas " style = "border:1px solid #000000;" >< /canvas>

Canvas Basico (15 de Abril del 2018) “elaboración propia”

Siempre se deb asignar un id al canvas para poder trabajar con el codigo JavaScript


Ahora,si se desea dibujar un rombo por ejemplo se debe trabajar de la siguiente forma:
Primero se le asigna un tamaño nuevo al canvas con los atributos width y height

< canvas id = "MiCanvas " width = "200" height = "100" style = "border:1px solid #000000;" >< /canvas>

Despues se utiliza el siguiente codigo JavaScript para empezar a dibujar. Se utiliza un sistema de coordenadas.

< script >
var can = document. getElementById ( "MiCanvas" );
var linea = can. getContext ( "2d" );
linea. moveTo ( 50 , 50 );
linea. lineTo ( 100 , 100 );
linea. moveTo ( 50 , 50 );
linea. lineTo ( 100 , 0 );
linea. moveTo ( 100 , 0 );
linea. lineTo ( 150 , 50 );
linea. moveTo ( 50 , 150 );
linea. lineTo ( 150 , 50 );
linea. stroke ();
< /script>
