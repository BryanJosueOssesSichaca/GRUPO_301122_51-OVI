Editores de Codigo

Publicado el 10/04/2018 a las 02:00

Los editores de código son programas que facilitan la gestion del código fuente de proyectos de desarrollo. Son ideales cuando se trabaja con diferentes lenguajes de programación, alternándolos o en un solo proyecto (por ejemplo, en un proyecto web es muy comun combinar html, css, js, php, ajax, etc ).
El código es texto y será interpretado como código cuando se ejecute en el contexto adecuado. Esto quiere decir que se puede escribir en cualquier entorno de texto plano como el bloc de notas en windows. A no ser que se realicen cambios superficiales no es comun trabajar en un proyecto ocupando estos entornos como el bloc de notas.
Los editores de codigo facilitan el trabajo, esto varia dependiendo el editor. Por ejemplo, son comunes las funciones de autocompletar, gestión de versiones, el cambio de colores en cada parte del texto según el contexto, la comprobacion de sintaxis dentro del código, la ejecucion directa de codigo a partir de plugins.


Recursos


En internet se encuentra mucha informacion, tops, evaluaciones sobre muchos editores de codigo que se encuentran disponibles hoy en dia, bien sea Open Source o aquellos que cuentan con una version gratuita y otra de pago con mas funciones.
Esimportante revisar en cada web consultada la fecha de esta, que no sea tan antigua porque todo se actualiza constantemente. Las webs esritas en ingles contienen la mejor informacion, las listas con las ventajas y desventajas de cada editor, brindan al futuro programador informacion sobre la mejor decision que puede tomar de acuerdo a sus requisitos.
En este O.V.i no se daran a conocer un monton de editores de codigo (porque si que hay bastantes), se busca mostrar a los estudiantes tres en particular que, segun el punto de vista del autor, se consideran buenos para los principiantes o aquellos que ya tienen experiencia en el mundo de la programacion. Se trataran:


- Notepad ++


- Brackets


- Sublime Text 3


Fueron elegidos teniendo en cuenta cuál es la mejor opción para alguien que desea iniciar hoy en día. El primero esta dedicado a principiantes. El segundo y el tercero, Brackets y Sublime Text 3, son recomendados especificamente para los que trabajan exclusivamente en lenguajes web.
