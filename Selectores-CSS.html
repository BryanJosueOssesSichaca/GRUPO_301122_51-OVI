Selectores CSS

Publicado el 15/04/2018 a las 00:37

Los selectores son una serie de patrones que se utilizan para indicar a que elemento HTML deseamos aplicar los estilos CSS definidos, existen multiples selectores en CSS los cuales podemos combinar para realizar selecciones mucho mas especificas y complejas.

Selector universal

Para seleccionar todos los elementos de una página solo se utiliza el símbolo * (asterisco), normalmente se utiliza para resetear los estilos predefinidos por los navegadores sin embargo cuando se combina con otros selectores, también puede hacer referencia a todos los elementos dentro de otro elemento.

* {

margin: 0;
padding: 0;
box-sizing: border-box;

}

Selector de etiqueta

Utilizando el nombre de una etiqueta HTML (p, div, img, span etc…) podemos aplicar estilos a todos los elementos de ese tipo concreto en el documento.

h1 {

/*codigo css*/

}

Podemos aplicar los mismos estilos a varias etiquetas también separándolas por coma (,)

h1, h2, h3, h4 {

/*codigo css*/

}

En el ejemplo los estilos serán aplicados a todas las etiquetas h1, h2, h3 y h4 dentro del documento.

Selector de Id

El nombre que indiquemos en el selector hará referencia al valor del atributo Id=” ” (el cual define un identificador único dentro del documento HTML que no debe de repetirse) de un elemento en concreto de la página, debemos de anteponer el símbolo # (numeral) al nombre del identificador.

#NombreDelIdentificador {

/*codigo css*/

}

Selector de clase

Todos los elementos html pueden hacer uso del atributo class=” ", la diferencia entre el atributo id=“ ” y es que el id tiene que ser único e irrepetible en todo el documento HTML mientras que un elemento HTML puede tener múltiples clases y estas clases pueden ser reutilizadas por otro elementos dentro del mismo documento también.

El nombre que indiquemos en el selector hará referencia al valor o uno de los valores del atributo class=” ” que posea un elemento HTML, debemos anteponer el símbolo . (punto) al nombre de la clase.

.NombreDeLaClase {

/*codigo css*/

}

Selector de atributos

Permite aplicar estilos a un elemento que hace uso de un atributo concreto y este atributo toma un valor determinado, debemos especificar el tributo y su valor entre [ ].

Con CSS3 disponemos de varias posibilidades con este tipo de selector:
Sintaxis Explicación
Elemento[atributo1=”valor”] selecciona todos los elementos que utilicen ese atributo y cuyo valor finalice con el valor indicado.
Elemento[atributo~=”valor”] selecciona todos los elementos que utilicen ese atributo y cuyo valor contenga el valor indicado.
Elemento[atributo^=”valor”] selecciona todos los elementos que utilicen ese atributo y cuyo valor inicie con el valor indicado.
Elemento[atributo*=”valor”] selecciona todos los elementos que utilicen ese atributo y cuyo valor contenga en cualquier parte (substring) el valor indicado.

Selectores jerárquicos

Los documentos HTML y XML están formados de manera jerárquica, lo que quiere decir que hay elementos que contienen a otros formando una estructura de árbol.

Menú "Options" Github Desktop (15 de Abril del 2018) “elaboración propia - Bryan Josue Osses Sichaca”

Por ejemplo el elemento <html> contiene al elemento <head> el cual a su vez contiene el elemento <title> lo que quiere decir que <head> es hijo directo de <html>, <title> a su vez es nieto de <html> y es hijo directo de <head>, los elementos que están al mismo nivel forman hermanos por ejemplo dentro de <head> esta <title> y a la vez la etiqueta <meta> lo que quiere decir que meta es hermano de <title> y también es hijo directo de <head>.

Teniendo en cuenta esta estructura jerárquica podemos crear selectores que encajen con cierta jerarquía dentro del documento.

Selectores Jerarquicos
Sintaxis Explicación
elemento1 elemento2 Los estilos son aplicados al elemento2 cuando es hijo del elemento1.
elemento1 > elemento2 Los estilos son aplicados al elemento2 cuando es hijo directo del elemento1.
elemento1 + elemento2 Los estilos son aplicados al elemento2 cuando es hermano del elemento1 y ademas el elemento1 precede al elemento2.
elemento1 - elemento2 Los estilos son aplicados al elemento2 cuando es hermano del elemento1 y le precede.
elemento:first-child Los estilos son aplicados cuando el elemento es el primer hijo de su padre.
elemento:last-child Los estilos son aplicados cuando el elemento es el ultimo hijo de su padre.
elemento:only-child Los estilos son aplicados cuando el elemento es el unico hijo.
elemento:nth-child(n) Los estilos son aplicados al elemento hijo numero n de su elemento padre.
elemento:last-child(n) Los estilos son aplicados al elemento hijo numero n de su elemento padre pero contando en orden inverso.

Ejemplos:

section article {

color: #000;
padding: 10px;
border: 1px solid #ccc;

}

los estilos CSS serán aplicados a los elementos article que sean hijos de los elementos section del documento.


header > h1 {

color: #000;
padding: 10px;
border: 1px solid #ccc;

}

los estilos CSS serán aplicados a los elementos header que sean hijos directos de los elementos header del documento.

Selectores Compuestos

Podemos combinar todos los selectores vistos anteriormente para hacer selecciones mucho mas específicas, por ejemplo podemos seleccionar los elementos p del documento cuyo padre sea un elemento article, cuya clase sea parrafo y que utilicen el atributo lang con el valor es.

article > p.parrafo[lang="es"] {

color: #f1f1f1;
font-size: 1em;

}

Pseudoclases

las pseudoclases permiten aplicar estilos CSS a un selector especifico cuando ocurre una determinada circunstancia (evento), algunas de las pseudoclases mas utilziadas son:

Pseudoclases
Pseudoclase Explicación
:focus se aplica cuando el elemento obtiene el foco.
:enabled se aplica cuando elemento esta habilitado.
:disabled se aplica cuando el elemento esta deshabilitado.
:checked se aplica normalmente a controles de formularios (input) de tipo radio o checkbox, cuando estén seleccionados .
:hover se aplica cuando el mouse o ratón pasa por encima del elemento
:active Se utiliza comúnmente en elementos <a> o elementos <button>, Los estilos son aplicados al elemento en el momento en que se da click sobre el mismo y se eliminan cuando suelta el botón del ratón.

algunas pseudoclases son solo aplicables a los enlaces <a>
Pseudoclase Explicación
a:link Los estilos CSS son aplicados a los enlaces que no han sido abiertos o visitados.
a:visited Los estilos CSS son aplicados a los enlaces que han sido visitados.

Ejemplos:

header:hover {

background: hsl(300, 100%, 90%);
border: 1px solid #660066;

}

Los estilos serán aplicados a los elementos header cuando el ratón pase por encima de los mismos.


form input[type="radio"]:checked + p {

background: #goldenrod;

}

Los estilos serán aplicados a los elementos p que sean hermanos de los elementos input con el atributo type="radio" los cuales preceden al elemento p cuando se marca o selecciona el elemento input.
