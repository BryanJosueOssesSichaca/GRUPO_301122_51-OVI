Bordes

Son los contornos de los elementos. Para establecer algunos bordes se deben definir propiedades adicionales.

  1. border-color

    :añade color al borde de los elementos.

Por ejemplo se añade borde a un parrafo

p {
border-style: solid ;
border-color: green ;
}

    : permite utilizar imágenes como bordes de los elementos de la página.

Por ejemplo se tiene la siguiente imagen y se desea colocarla como borde.

Color

  1. HSL colors

    : un color se peude indicar usando tono, saturacion y luminosidad.
H
Hue o Tono 		El tono es un grado en la rueda de color de 0 a 360. 0 es rojo, 120 es verde y 240 es azul.
S
Saturation o Sturacion 		Es un valor porcentual donde: 0% significa un tono de gris y 100% es el color completo.
L
Lightness o Luminosidad 		Es un porcentaje donde: 0% es negro, 50% neutro y el 100% es blanco.

Se relaciona codigo html-css para trabajar con los colores en formato HSL

  1. text-overflow

    : permite controlar el contenido desbordado y que no se muestra. Puede ser cortado, mostrar una elipsis ('…') o una cadena de texto personalizada.

Funciona de la siguiente manera:

div. . a {
/*No ajusta el texto al control*/
white-space : nowrap ;
/*Ancho del elemento*/
width : 100px ;
/*Desbordamiento de texto oculto*/
overflow : hidden ;
/*Lo que se meustra al usuario para indicarle la falta de texto*/
text-overflow : ellipsis ;
/*Borde del elemento*/ border : 1px solid #000000 ;
}

/*evento hover del mouse*/
div.contenedor1 div. . a : hover {
/*Se muestra el texto al pasar el mouse*/
overflow : visible ;
}

Se muestra un elemento con el texto incompleto y al pasar el mouse se muestra enh su totalidad.
