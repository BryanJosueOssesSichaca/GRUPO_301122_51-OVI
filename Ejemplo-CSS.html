La primera impresion da mucho que decir de algo. Todo entra por los ojos y lo procesa el cerebro. Esto mismo se aplica al desarrollo web, si una pagina web es simple, no tiene colores, los botones son estaticos y con el contenido dan ganas de salir de ahi lo mas pronto posible, es seguro que no se esta haciendo una buena gestion y esto llevara al fracaso, pero en cambio, si la pagina esta bien estilizada, con colores bonitos y apariencia agradable, los usuarios se sentiran a gusto con ella y no querran salir de ella, con lo que se asegura exito rotundo.


CSS se dedica a eso y mucho mas, la apariencia lo es todo y este lenguaje de Hojas de estilo en cascada brinda infinidad de elementos que pueden ocupar los programadores para dar a concocer al mundo un proyecto web de maravilla. CSS trabaja de la mano con HTML porque ¿como pintas, amueblas y decoras una casa si esta no esta construida?, esta es una analogia valida para dar a entender lo que se pretete explicar.
CSS ha evolucionado el mundo de desarrollo web, ya en su tecera version, cuenta con mucho terreno por recorrer para nunca dejar de soprendernos.
