Media Query CSS3

Publicado el 15/04/2018 a las 21:00

Las media queries (Media Query) son un módulo introducido en CSS3 el cuales nos permite definir estilos CSS que se aplicarán al documento si se cumple cierta condición o circunstancia.

Son la base fundamental del responsive web design, método que se ha convertido en un estándar a la hora de diseñar sitios web actuales ya que nos permite diseñar sitios web que puedan adaptarse a la resolución y tamaño de pantalla de los diferentes dispositivos (moviles, televisores, diferentes resoluciones de pantalla de computadores etc...) en los cuales se visualiza la pagina web.

Las media quieres actúan como un bloque de reglas CSS que serán aplicadas sobre el documento HTML cuando se cumpla cierta condición, siempre comienzan con la palabra reservada o instrucción “@media” seguido del primer parámetro “mediatype” (el cual indica el tipo de medio al que las reglas CSS serán aplicadas).

mediatype
Nombre Explicación
All Aplicable a documentos impresos o medios paginados.
Print Aplicable a documentos impresos o medios paginados.
Screen Aplicable primeramente a pantallas de computadores.
TV Aplicable dispositivos televisores.
handheld Aplicable a dispisitivos moviles (pantalla pequeña con poco ancho de banda).
Speech Aplicable a sintetizadores de voz.

Y el segundo parámetro el “media feature” donde especificamos al navegador que condición debe cumplir el dispositivo para que se apliquen los estilos CSS definidos en la media query (los mas comunes se refieren a las dimensiones de la pantalla del dispositivo height, width, max-height, max-width, min-height, min-width y tambien los que se refieren a la orientación del dispositivo orientation landscape o portrait).

Las media queries pueden utilizar multiples operadores lógicos en su declaración, los más utilizado son and y only.

  1. And – se utiliza para colocar juntas multiples “media features” a ser evaluadas.
  2. Only – previene que los navegadores antiguos que no soportan las media queries apliquen los estilos asignados.

Ejemplo Media Query

@media screen and (max-width: 650px) {

/* Codigo CSS */

}

Esta media query solo se aplica si el mediatype concuerda con screen y si el ancho maximo de la pantalla son 650px.

@media handheld and (max-width: 650px) and (orientation: landscape){

/* Codigo CSS */

}

Esta media query solo se aplica si el mediatype concuerda con handheld y si el ancho maximo de la pantalla son 650px y si la orientación de la pantalla es vertical.

Existen 2 maneras de declarar media queries:

1. a la hora de enlazar un archivo CSS con la etiqueta link podemos utilizar el atributo media=” ”

<link rel="stylesheet" media="only screen and (max-width: 500px)" href="estilos.css">

2. dentro de un archivo CSS o etiqueta style.

La diferencia radica en que si definimos la media query a la hora de enlazar un archivo CSS solo podremos definir 1 la cual será aplicada a esa hoja de estilos en concreto, en cambio dentro de una hoja de estilos .css podemos definir múltiples media queries.
