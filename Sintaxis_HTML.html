La sintaxis en el mundo del desarrollo se refiere a la forma en que está escrito el código fuente, es muy variada, depende del lenguaje sobre el cual se esté trabajando. Algunas pueden ser similares, otras completamente diferentes y otras confunden con sus nombres, por ejemplo: Java y JavaScript, aunque ambos tengan la palabra "Java" no quiere decir que estos lenguajes presenten similitudes, al contrario, son muy diferentes.


HTML se basa em marcas o etiquetas. Estas permiten definir la estructura del documento, por ejemplo: títulos, bloques de texto, videos, imágenes, enlaces, etc. Así, se indica al navegador que contenido y como lo debe mostrar al ser interpretado.
Además, incorpora sintaxis de otros lenguajes, en caso de tratarse de un documento muy básico, por ejemplo, en una página escrita en HTML podemos encontrar CSS en la cabeza de la página y JavaScript al final de la página y serán interpretados sin ningún problema.


Un aspecto interesante de este lenguaje es que es fácil de aprender, no se requiere tener conocimientos avanzados de desarrollo web antes de ponerse en marcha a codificar.
