Concepcion

CSS puede ser más que un lenguaje de hojas de estilo personalizadas, aparte de darle apariencia a nuestra página web podemos implementar muchísimas cosas que sin lugar a duda no se pensaba que se pudieran hacer: juegos, animaciones y efectos combinados que antes solo se conseguía con otro JavaScript y otros lenguajes. Por ejemplo se han creado: un minion, letreros de neon, figuras en tercera dimensión y dibujos que las personas al verlos no se imaginan que esta detrás una hoja de estilos. Deja que tu imaginación vuele y tú mismo te pones los limites en este mundo de la programación.

Uso

Puede incorporarse de tres formas


Inline

Cuando se desea efectuar cambios sobre un elemento en específico. Se coloca dentro de la etiquetas de apertura del elemento.

< ! - - Se alinea el texto contenido en el titulo- - >
< h1 style = "text-align:center" ; > Titulo < h1/>

Resultado:

Titulo


< ! - - Se cambia el color del contenido de la línea de texto- - >
< p style = "color:red; text-align:center;" > Este es una línea de texto modificada con css < /p>

Resultado:

Este es una línea de texto modificada con css


Internal

Cuando se desea realizar cambios a nivel de documento HTML. Se indica entre las etiquetas < head > < /head > colocando el código dentro de las etiquetas < style > < /tyle >
Por ejemplo:

< style >
< ! - - asigna color de fondo al cuerpo de la página - - >
body { font-size: 1.3em ;
}

< ! - - asigna tamaño de letra a los párrafos de la página - - >
p { background-color: blue ;
}
< /style >


External

Cuando se desean hacer cambios globales en todos los archivos del proyecto. Se escribe en un documento con extensión .css y se llama entre las etiquetas < head > < /head >
Por ejemplo: se desea realizar cambios en la cabecera, secciones, artículos y pies de página, para estandarizar el estilo en las páginas de nuestra web.

< ! - - Este es mi documento externo .css - - >
header
{ background-color: #C5C9FB ;
font-size: 15 px ;
text-underline-position: all ;
}

section, article {
font-weight: 75 % ;
font-size: 2 em ; }

footer {
font-size: 10 px ;
font-weight: bold ;
}


Se llama de la siguiente manera:

< head >
< link rel = "stylesheet" type = "text/css" href = "hojadeestilos.css" />
< /head >

En el atributo href
entre las comillas se indica la ruta del archivo.
