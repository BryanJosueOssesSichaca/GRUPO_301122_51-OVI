Configuración inicial de GIT

Publicado el 06/04/2018 a las 00:45

Una vez que esté instalado git y si es la primera vez que lo utilizamos, tenemos que configurar nuestros datos como nombre, correo electornico, editor etc… para esto utilizamos el comando config en la línea de comandos (GIT bash para Windows, la terminal del sistema para distribuciones de Linux y Mac Os):

Configurar nombre de usuario:

$ git config --global user.name “NombreDeUsuario”

Configurar Correo Electronico:

$ git config --global user.email CorreoElectornico@ejemplo.com

GIT siempre utilizará esta informacion para todo lo que hagas en el Sistema, para sobrescribir los datos solamente hay que utilizar el mismo comando pero con un nombre o email diferente.

Configurar Editor

Puedes especificar el editor que GIT utilizará por defecto cuando GIT necesite que introduzcas un mensaje (como la descripción de los commits) en caso de no configurar esta opción git utilizara el editor por defecto del sistema.

$ git config -- global core.editor NombreEditor

Comprobar la configuración

Para comprobar La configuración utilizamos el comando:

$ git config -- global --list

Configuración de Git a través de Github Desktop

Si utilizas GIT a través de una herramienta que nos proporcione una interfaz gráfica como Github Desktop, GIT kraken o source tree no tendrás que ingresar ningún comando de manera manual sino acceder a través de los menús de opciones.

En Github Desktop debemos ir al menú File -> Options, allí nos encontraremos multiples pestañas:

Accounts:

Aquí podemos iniciar sesión con nuestra cuenta de Github (Usuario y contraseña).

Git:

igual que con el comando config, podemos configurar nuestro nombre y correo electrónico.

Advanced:

Podemos elegir el editor externo, el Shell para la línea de comandos de GIT (por defecto GIT bash) entre otras opciones.
