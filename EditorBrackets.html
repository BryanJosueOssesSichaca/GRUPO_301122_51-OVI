El editor de codigo Brackets esta diseñado para programadores mas experimentados con enfasis en desarrollo web. Esta desarrollado en HTML, CSS y JavaScript, es de codigo abierto (hace parte del proyecto Open Source), recien instalado el tamaño en disco ocupado es aproximadamente de 185Mb y cuenta con soporte para varios lenguajes de programacion. En cuanto a su apariencia, es mas agradable a la vista y con una interfaz sencilla para el programador. Su logo lo representan dos parentesis o brackets que son uno de los simbolos principales ocupados en la programacion.
Principales Caracteristicas
- Vista Previa Dinamica: permite visualizar en tiempo real el codigo escrito.
- Multiples Extensiones: en colaboracion con GitHub cuenta con multiples extensiones para facilitar el desarrollo en proyectos.
- Edicion Rapida: muestra sugerencias de determinado lenguaje y se aplica a varias funciones y propiedades a la vez.
- Flexible por su amplio soporte para los diferentes lenguajes de programacion.
- Vista Dividida: permite trabajar dos archivos al mismo tiempo aunque sean completamente diferentes.
- Paleta de colores: para no estar buscando los codigos de los colores que se requieran.
