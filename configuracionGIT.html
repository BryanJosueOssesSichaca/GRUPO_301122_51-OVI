Configuración inicial de GIT

Publicado el 06/04/2018 a las 00:45

Una vez que se haya instalado git y si es la primera vez que se utiliza, se configuran los datos básicos: nombre, correo electrónico, editor etc… para esto, se ocupa el comando config en la línea de comandos (GIT bash para Windows, la terminal del sistema para distribuciones de Linux y Mac Os):

Configurar nombre de usuario:

$ git config --global user.name “NombreDeUsuario”

Configurar Correo Electrónico:

$ git config --global user.email CorreoElectornico@ejemplo.com

GIT siempre utilizará esta información para todas las acciones realizadas en el Sistema, para sobrescribir los datos solamente hay que utilizar el mismo comando pero con un nombre o email diferente.

Configurar Editor

Se puede especificar el editor que GIT utilizará por defecto cuando GIT cuando requiera que sea introducido un mensaje (como la descripción de los commits) en caso de no configurar esta opción git utilizara el editor por defecto del sistema (en Windows, Git Bash utiliza por defecto el editor Vim).

$ git config -- global core.editor NombreEditor

Comprobar la configuración

Se utiliza el siguiente comando:

$ git config -- global --list

Configuración de Git a través de Github Desktop

Si se utiliza GIT a través de una herramienta que proporcione una interfaz gráfica como Github Desktop, GIT Kraken o Source Tree, no se requiere ingresar ningún comando de manera manual sino acceder a través de los menús de opciones.

En Github Desktop al dirigirse al menú File -> Options, se aprecian múltiples pestañas:

Accounts:

Para iniciar sesión con la cuenta de Github (Usuario y contraseña).

Git:

Igual que con el comando config, se puede configurar el nombre y correo electrónico.

Advanced:

Para elegir el editor externo, el Shell para la línea de comandos de GIT (por defecto GIT bash) entre otras opciones.

Obtener ayuda sobre un comando

En Git para visualizar la página del manual (manPage) donde se encuentra información sobre cualquier comando, se realiza siguiendo la sintáxis:

$ git help NombreDelComando

$ git NombreDelComando --help
