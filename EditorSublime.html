El editor de codigo Sublime Text 3 es muchos mas sofisticado que los anteriores mencionados. Esta desarrollado en C++ y Python, cuenta con una version prueba bastante funcional y sin limite de tiempo y se peude adquirir la version completa registrandose y pagando una licencia, recien instalado el tamaño en disco ocupado es aproximadamente de 24.7Mb y cuenta con soporte para varios lenguajes de programacion. En cuanto a su apariencia, es agradable a la vista y con una interfaz libre de distracciones para el programador. Su logo tiene se asemeja a un rayo por ser muy rapido al momento de ejecutarse y gestionar las diferentes tareas.
Principales Caracteristicas
- Versatilidad: permite tener multiples documentos abiertos.
- Navegacion Lateral: para recorrer el codigo de una manera mas facil y comnoda.
- Multiples Cursores: facilita la edicion de fragmentos de codigo iguales en diferentes lineas.
- Autocompletado: para una codificacion mas rapida muestra sugerencias y parametros de las etiquetas ocupadas.
- Carpeta de archivos: permite abrir en una nueva ventana una carpeta con varios archivos de diferentes extensiones y navegar entre estos a traves de un menu lateral.
- Vista Dividida: permite modificar el entorno de desarrollo para trabajar con varias ventanas.
