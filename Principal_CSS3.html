CSS3

Escrito el 05/04/2018 a las 11:00

El nombre viene de Cascading Style Sheets y sus siglas CSS, que en español traduce Hojas de estilo en cascada. Este es un lenguaje de programación usado para definir la presentación de un documento estructurado escrito en HTML o XML (y por extensión en XHTML), y con esto lo que se persigue es la idea de separar la estructura de un documento de su presentación.
La información de estilo puede ser adjuntada como un documento separado o en el mismo documento HTML. En este último caso podrían definirse estilos generales en la cabecera del documento o en cada etiqueta particular mediante el atributo " <style>".

Para qué sirve el CSS3?

• Css sirve para personalizar una página de un sitio web en un documento externo y esto permite que en la Hoja Css podamos cambiar la estética de un sitio web pero esto quiere decir que podemos variar el estilo sin tocar el lenguaje HTML que compone el Css.
Cuando nos referimos a estética de la página damos referencia a lo que son colores, tamaños de las fuentes, tamaños de elemento con css podemos establecer diferente reglas para mejorar la estética del elemento, podemos indicar por ejemplo como debe presentarse un documento. Podemos indicar propiedades como el color, el tamaño de la letra, el tipo de letra.

Algunas características de CSS3

• Atributo gradiente de colores en borde con CSS y Firefox: Es una posibilidad de definición de color en los bordes de los elementos de CSS con un atributo no estándar.
• Múltiples imágenes de fondo con CSS: con CSS básico y con características de CSS 3.
• Bordes con imágenes en CSS 3: El atributo border-image harán posible la utilización de imágenes como bordes de los elementos de la página, sin código HTML especial, simplemente con hojas de estilo.
• Propiedad background-origin de CSS 3: La propiedad background-origin permite decidir la posición de la imagen de fondo con respecto al borde, padding o el contenido del elemento.
• Sombras en el texto con text-shadow de CSS: Cómo aplicar sombras y otros efectos en los textos con CSS y el atributo text-shadow.

Caracteristicas Principales.



Múltiples imágenes de fondo


Logra que un elemento de la página tenga varias imágenes de fondo a la vez.


Bordes con imágenes


Hace posible la utilización de imágenes como bordes de los elementos de la página, sin código HTML especial, simplemente con hojas de estilo.


Bordes redondeados


Define la curvatura que debe tener el borde del elemento.


Word-wrap


Esta propiedad sirve para romper las palabras que son demasiado largas y no caben enteras por la anchura de una caja.


Textos multi-columna


Permite maquetar directamente el texto en varias columnas.


Sombras


Permite aplicar sombras a los elementos de la página, sin usar imágenes, Javascript ni nada extra, simplemente con un atributo de CSS3.

Ventajas y desventajas de css3

Ventajas

• Se obtiene un mayor control a la hora de la presentación de la página aún más teniendo todo el código Css reunido en uno para facilitar la modificación. Cuando elegimos un archivo con el código Css aumenta la accesibilidad ya que con facilidad podemos asignar un código Css concreto a personas con alguna incapacidad visual.
